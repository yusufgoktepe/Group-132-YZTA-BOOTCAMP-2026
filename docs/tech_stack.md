# Tech Stack

| Alan | Teknoloji |
|---|---|
| Mobil Uygulama | React Native / Expo |
| Backend | FastAPI |
| Veri Tabanı | SQLite / PostgreSQL |
| AI / Data Science | Python, pandas, scikit-learn |
| Proje Yönetimi | GitHub Projects, GitHub Issues |

## Neden Bu Teknolojiler?

- React Native / Expo: Mobil-first ürün için hızlı prototipleme.
- FastAPI: Python ve AI/ML entegrasyonu için uygun backend yapısı.
- SQLite: İlk geliştirme aşamasında hızlı ve kolay başlangıç.
- PostgreSQL: Proje büyüdüğünde daha güçlü veri tabanı seçeneği.
- scikit-learn: İlk öneri sistemi prototipi için sade ve anlaşılır modelleme.
